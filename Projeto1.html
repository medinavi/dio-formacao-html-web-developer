Estrutura básica do HTML

(Voltar ao topo)

O HTML (HyperText Markup Language) é a linguagem usada para criar páginas da web. Ele utiliza uma estrutura básica composta por três partes principais: as tags HTML, head e body.

  1. Tag HTML
    2. Toda página HTML começa com uma tag "html" . Essa tag marca o início do documento HTML e também o seu fim. É como se fosse o invólucro que envolve todo o conteúdo da página.

  2. Tag head
    3. Representada pela tag "head", esta seção não é visível na página, mas é onde você coloca informações importantes sobre o documento, como o título da página, links para estilos CSS, metadados e até mesmo scripts JavaScript. O conteúdo da tag head é crucial para mecanismos de busca e navegadores, mas não é mostrado diretamente aos visitantes.

  3. Tag body
    4. A parte visível da página é definida dentro da tag "body". É aqui que você coloca todo o conteúdo que deseja que os visitantes vejam, como textos, imagens, links e vídeos. Tudo o que estiver dentro da tag body será exibido na janela do navegador.

Atributo das Tags

(Voltar ao topo)

Tags em HTML são elementos fundamentais para estruturar e formatar conteúdo em páginas da web. Elas são cercadas por colchetes angulares e podem conter propriedades que definem como o elemento deve ser exibido ou se comportar. Aqui estão alguns exemplos de tags e suas propriedades mais comuns:

  • Tag img
    • Propriedades principais: src, alt, width, height. A tag img é usada para incorporar imagens em uma página da web. As propriedades, como src (URL da imagem), alt (texto alternativo) e dimensões (largura e altura), são frequentemente especificadas para controle visual e acessibilidade.

  • Tag input (com o atributo type)
    • Propriedades: type, name, value. A tag input é usada para criar campos de entrada em formulários. O atributo type define o tipo de entrada, como texto, caixa de seleção, botão de rádio, etc. Outras propriedades, como name e value, são frequentemente usadas para identificar e preencher os campos do formulário.

  • Tag com id
    • Propriedade: id. O atributo id é usado para atribuir um identificador único a um elemento HTML. Isso permite que você selecione e estilize o elemento com CSS ou faça referência a ele usando JavaScript para manipulações dinâmicas.

Textos

(Voltar ao topo)

As tags usadas para formatar e exibir texto são essenciais para criar conteúdo legível e atraente em páginas da web. Aqui estão algumas das tags mais comuns usadas para texto em HTML:

  • Tag p (parágrafo):
    • A tag p é usada para definir parágrafos de texto. Ela cria uma quebra de linha antes e depois do conteúdo do parágrafo, tornando o texto bem formatado.

  • Tag h1, h2, h3, h4, h5, h6 (títulos e subtítulos)
    • Essas tags são usadas para criar títulos e subtítulos em uma página. h1 é o título principal e h6 é o menor subtítulo. Eles são frequentemente usados para estruturar o conteúdo da página.

  • Tag strong (texto em negrito)
    • A tag strong é usada para enfatizar o texto, tornando-o visualmente mais forte. O texto dentro dessa tag é geralmente exibido em negrito pelo navegador.

  • Tag i (texto em itálico)
    • A tag i é usada para formatar o texto em itálico.

  • Tag blockquote (citação em bloco)
    • A tag blockquote é usada para criar citações em bloco, destacando trechos de texto que são retirados de outra fonte. Geralmente, o navegador exibe o texto em um recuo ou de alguma forma diferente para indicar que é uma citação.

  • Tag sup (texto sobrescrito)
    • A tag sup é usada para criar texto sobrescrito, que é exibido acima da linha de texto normal. Isso é frequentemente usado para índices, notas de rodapé e fórmulas matemáticas.

  • Tag mark (marcação de texto)
    • A tag i é usada para formatar o texto em itálico.

Listas ordenadas e não ordenadas

(Voltar ao topo)

As Listas ordenadas são representadas pela tag "ol" e enumera ordenadamente os itens de uma lista. Para formar as listas é necessário utilizar a tag "li" (chamadas de tags filhas) dentro da tag de lista ordenada (ol).

Já as Lista não ordenadas são representadas pela tag "ul" e trás marcadores ao invés de numerar os itens da lista. Também se utilisa de tags filhas para funcionar.

Links

(Voltar ao topo)

Links em HTML são elementos que permitem aos usuários navegar de uma página da web para outra, seja dentro do mesmo site ou para sites externos. Eles são criados usando a tag "a", que é a abreviação de "âncora". A tag "a" permite que você crie hiperlinks, que são textos ou elementos clicáveis que redirecionam o usuário para outra página da web quando são clicados.

  • Tag a
    • Esta é a tag de âncora que define o início do link.

  • href="URL_do_destino"
    • O atributo href (hiperlink de referência) especifica o URL do destino para o qual o link apontará. Você precisa substituir "URL_do_destino" pela URL real da página ou recurso para o qual deseja criar um link.

  • Texto entre as tags "a"e "/a"
    • O texto que você coloca entre as tags "a" e "/a" é o que será exibido como o texto clicável para o usuário. Isso pode ser qualquer texto ou até mesmo uma imagem, se você preferir.

Os links são uma parte fundamental da experiência de navegação na web, e é importante que eles sejam descritivos e relevantes para os usuários. Além disso podemos incluir atributos adicionais à tag "a" para controlar como o link se comporta, como target para abrir em uma nova janela ou guia, ou title para fornecer informações adicionais quando o usuário passa o mouse sobre o link.
